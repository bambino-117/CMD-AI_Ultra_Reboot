INITIALISATION REQUISE

Bienvenue, Opérateur. Le système a détecté un premier lancement. Une configuration initiale est nécessaire pour crypter votre identité et sécuriser votre espace de travail personnel, le "Bunker".

Veuillez suivre les instructions pour établir le protocole.

PROTOCOLE D'IDENTITÉ

Créez votre identifiant unique et votre clé d'accès. Ces informations seront utilisées pour sécuriser l'accès à vos données.

CONFIGURATION DU BUNKER

Le Bunker est votre espace de stockage chiffré. Confirmez ou modifiez le chemin d'accès sur votre système hôte.

CONFIGURATION TERMINÉE

Le protocole d'initialisation est terminé. Votre identité est sécurisée et le Bunker est prêt. Le système va maintenant redémarrer avec vos nouveaux paramètres.

Préparez-vous à la connexion.
